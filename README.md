# Code to replicate main findings from:
Sabat, M., Gouyette, H., Gaucher, Q., Espejo, M. L., David, S. V., Norman-Haignere, S., & Boubenec, Y. (2025). Neurons in auditory cortex integrate information within a constrained and context-invariant temporal window. Current Biology, 35(24), 6114-6125.e7. https://doi.org/10.1016/j.cub.2025.11.011


## Repository Structure

The code in this repository assumes a **fixed directory structure**. Please ensure that data and code are organized exactly as described below.

- All **code** must reside in the `code/` directory.
- All **data** must be placed in the `data/` directory.
- The `code/` and `data/` directories must be at the **same directory level**.

## Data Availability

The data used in this study are available via the Open Science Framework (OSF): https://doi.org/10.17605/OSF.IO/HVY9B.
Please download the data from the OSF link provided in the paper and place the contents directly into the `data/` directory without modifying the internal structure.

## Usage Notes

- Paths in the code are defined relative to the project root and assume the directory structure shown above.
- Changing directory names or nesting levels may cause the code to fail unless paths are updated accordingly.
